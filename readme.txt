Задача на создание учебного приложения с использованием gRPC и Go
Описание задачи:
Вам необходимо разработать учебное приложение для обмена сообщениями между пользователями. Приложение должно использовать gRPC для обеспечения межсервисного взаимодействия.

Требования:
Создайте два сервиса: "Сервис отправки сообщений" и "Сервис получения сообщений".

Каждый сервис должен иметь следующие методы:

Отправка сообщения: Пользователи могут отправлять сообщения друг другу. Метод должен принимать отправителя, получателя и текст сообщения.
Получение сообщений: Пользователи могут получать свои сообщения. Метод должен принимать идентификатор пользователя и возвращать список сообщений.
Используйте gRPC для определения сервисов и методов.

Создайте клиентское приложение, которое позволяет пользователям отправлять сообщения и читать свои сообщения.

Разработайте простой механизм аутентификации, чтобы идентифицировать пользователей.

Все данные (пользователи, сообщения и их состояние) могут храниться в памяти для упрощения.

Дополнительные задания (по желанию):
Добавьте возможность отправки сообщений множеству пользователей (групповые сообщения).

Реализуйте хранение данных в базе данных для долгосрочного сохранения сообщений и пользователей.

Разработайте простой интерфейс веб-приложения для взаимодействия с сервисом отправки сообщений.

Обеспечьте безопасность и аутентификацию с использованием токенов или сертификатов.

При выполнении этой задачи вы получите опыт работы с gRPC и Go, а также разработаете полезное учебное приложение для обмена сообщениями. Убедитесь в том, что вы импортируете и используете необходимые библиотеки для gRPC в вашем проекте.
